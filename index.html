async def lifespan(app: FastAPI):

FARA CRM is built on the Apps Architecture pattern ‚Äî each module is an independent application (App) with its own models, routers, access rights, and dependencies.

Environment ‚Äî the central facade: contains Apps, Models, and Settings. Monkey-patch trick allows extending models via ExtensibleMixin and @extend decorator without modifying source code.

Three component types: App (business logic + CRUD + ACL), Service (infrastructure: DB, logger, cron), and Mixin (extensions: Google Drive, Telegram, WhatsApp ‚Äî plug into existing models).
